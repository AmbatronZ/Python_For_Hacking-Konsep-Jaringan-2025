# Tata cara menjalankan Code #
 1. buat code/masukan code sesuai dengan file diatas
 2. masukan IP anda di server dan di socketnya di dalam code, cara mengecek IP ketikan saja di terminal ipConfig
 3. lalu copy dan paste kan saja IP kalian kedalam code
 4. jalankan Server lalu socket nya di terminal
 5. ketik di terminal : *Python3 UdpServer.py* untuk menjalankan server
 6. jika berhasil maka akan ada output **server up** (sesuai dengan input yang dimasukan sebagai tanda servernya nyala di codingan)
 7. selanjutnya ketikan lagi diterminal : *python3 UdpSocket.py* untuk menjalankan socket nya agar bisa mengirim dan menerima data
 8. jika berhasil akan muncul Output **Pesan dari server: "Selamat datang di UDP server"** (Sesuai dengan pesan yang di input kan)
